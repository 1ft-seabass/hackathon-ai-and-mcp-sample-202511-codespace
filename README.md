# ハッカソン向け ChatGPT API と MCP のサンプル集

> 2025年11月時点での田中正吾による、ハッカソンでザっと作れるようにするための説明用の資料とサンプル群です。

## このリポジトリについて

ChatGPT API を使った開発と、MCP (Model Context Protocol) を活用したエージェント構築のサンプルを集めたリポジトリです。

## どこから始めるか

1. **技術選択**: まず [00_technology-guide](./00_technology-guide/) で使う技術を選ぶ
2. **環境構築**: `01_setup/` で開発環境を整える（準備中）
3. **基礎から学ぶ**: `02_api-basics/` から順に試す（準備中）

ハッカソンですぐ使いたい方:
- シンプルなツール実行: `03_function-calling/`（準備中）
- 複雑なエージェント: `07_mcp-api-wrapper/`（準備中）

## 学習パス
